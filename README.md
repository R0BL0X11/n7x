# N7x - Flight and High Jump System

نظام الطيران والقفز العالي لـ Roblox

## الميزات ✨

✅ **واجهة مستخدم (GUI)** - زر سهل الاستخدام  
✅ **نظام الطيران** - اطير بحرية كاملة  
✅ **القفز العالي** - اقفز بارتفاع عالي جداً  
✅ **تشغيل/إيقاف** - اضغط على الزر لتفعيل أو تعطيل  

## كيفية التثبيت 🚀

1. في Roblox Studio، افتح **StarterPlayer**
2. انقر بزر اليمين على **StarterPlayer** → **Insert Object** → **LocalScript**
3. انسخ محتوى ملف `FlightSystem.lua` والصقه في السكريبت
4. اختبر المشروع!

## التحكم ⌨️

### الزر الرئيسي:
- اضغط على الزر الأزرق في أعلى الشاشة لتفعيل/تعطيل النظام

### أثناء الطيران:
- **W** - للأمام
- **S** - للخلف
- **A** - يسار
- **D** - يمين
- **Space** - أعلى
- **Ctrl** - أسفل

## الحالات 🎮

- **تشغيل** (أخضر) - النظام مفعل ✅
- **إيقاف** (أزرق) - النظام معطل ❌

## التخصيص ⚙️

يمكنك تعديل هذه القيم في السكريبت:

```lua
local flySpeed = 50      -- سرعة الطيران
local jumpPower = 100    -- قوة القفز
```
